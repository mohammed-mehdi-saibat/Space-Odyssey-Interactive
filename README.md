# Space Odyssey - Version Interactive

## Contexte du Projet

Vous faites partie de l’équipe Front-End chez **AstroTech Studio**, une entreprise spécialisée dans la création de plateformes éducatives sur l’espace et les sciences.

Après avoir conçu et intégré le site **Space Odyssey** en HTML/CSS, l’entreprise souhaite désormais créer une **version interactive**, intégrant des fonctionnalités dynamiques grâce à JavaScript.  
L’objectif est de rendre le site plus vivant, fonctionnel et engageant pour les utilisateurs, en ajoutant des interactions modernes : validation de formulaires, recherche, filtres, gestion de favoris, et plus encore.

---

## Objectifs du Projet

L’objectif principal est d’améliorer le site existant **Space Odyssey** en y intégrant des fonctionnalités JavaScript pour :

- Dynamiser l’expérience utilisateur.
- Manipuler le **DOM (Document Object Model)** pour afficher, modifier et supprimer des éléments.
- Mettre en pratique la gestion d’événements, le stockage local et la validation des formulaires.
- Comprendre les bases des opérations **CRUD (Create, Read, Update, Delete)** côté front-end.

---

## Fonctionnalités Implémentées

### Validation de Formulaires

- Vérification que tous les champs requis sont remplis.
- Validation du format d’email.
- Messages d’erreur clairs sous les champs invalides.
- (Bonus) Changement dynamique de style si le champ est valide ou invalide.

### Recherche et Filtrage Avancés

- Barre de recherche permettant de filtrer les missions spatiales par :
  - Nom
  - Agence
  - Objectif
  - Date de lancement
- Filtres multiples : par agence, par année, par type de mission.
- Filtrage combiné (texte + filtres).
- (Bonus) Affichage dynamique sans recharger la page.

### Opérations CRUD Complètes

Sur la page **Missions Spatiales** :

- **Ajout** : via un formulaire modal pour créer une nouvelle mission.
- **Édition** : modification des missions existantes directement dans l’interface.
- **Suppression** : confirmation avant la mise à jour du DOM.
- **Affichage** : cartes responsives automatiquement générées via le DOM.

### Système de Favoris

- Bouton “Ajouter aux favoris” sur chaque mission.
- Section dédiée aux favoris, avec possibilité de retirer les missions.
- Persistance des favoris via **localStorage**.

### Gestion d’Événements

- Clics sur les cartes, boutons ou liens.
- Changement d’onglets : "Toutes les missions", "Favoris", "Mes missions".
- Soumission, validation et réinitialisation de formulaires.
- Interaction fluide et sans rechargement de page.

---

## Bonus et Recommandations

- Validation avancée avec **regex** pour email, dates, etc.
- Simulation de vérification asynchrone de données externes.
- Améliorations UX : suggestions dynamiques, filtres à facettes multiples, animations de transition des cartes.
- Fonctionnalités premium possibles : export des favoris en PDF, partage via lien dynamique, système de notation.
